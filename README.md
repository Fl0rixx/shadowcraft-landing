# ShadowCraft Landing Page

Современный адаптивный лендинг для Minecraft RPG Survival сервера.

## Возможности

- Адаптивная верстка
- Современный тёмный дизайн
- Галерея игровых локаций
- Блок преимуществ сервера
- Карточка сервера с IP и онлайном
- Копирование IP в один клик
- Плавная навигация по странице

## Технологии

- HTML5
- CSS3
- JavaScript
